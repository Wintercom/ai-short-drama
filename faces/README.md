# 角色画像目录（faces/）

把你自己的角色画像放在这里，让 AI 短剧用「指定的演员」贯穿整部剧——
后续所有镜头的图生图、图生视频都以这张画像为参考图，保证跨镜头同一张脸。

## 两种指定方式（任选其一）

### 方式 A：剧本里写路径（优先级最高）
角色行第 5 段写画像路径（相对剧本文件所在目录）：

```
## 角色
- 林夏 | 性格 | 外貌 | 女 | faces/林夏.png
```

### 方式 B：按角色名放同名图（本目录）
把画像命名为「角色名.扩展名」放进本目录，程序自动匹配：

```
faces/
  林夏.png
  陈默.jpg
```

支持扩展名：`.png` / `.jpg` / `.jpeg` / `.webp`
目录可用 `FACES_DIR` 环境变量自定义（默认 `faces`）。

## 优先级与降级
显式路径（A） > 按名匹配（B） > AI 自动生成锚点（都没有时）。
指定的图找不到会打印警告并回退 AI 生成，**流程不中断**。

## 生效条件
参考图驱动的「同一张脸」需要 `T2I_PROVIDER=wanedit`（通义千问图像编辑）。
其它 T2I（pollinations / local）下，画像仅作弱参考或不生效。

> 注意：本目录下的图片是你的私有素材，已在 .gitignore 中忽略，不会被提交。
